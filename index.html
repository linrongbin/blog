Why：3 个平台都支持，很猛，reduce learning curve
Version: Help–> About

vscode extensions

How: Extensions –> Search hexo –> vscode-hexo

Git Config

  1. Repository Setup
    To create a repository named with “blog”
    Copy this： https://github.com/{yourself}/blog.git
  2. Enable GitPage
    a. Go to “Setting” link
    b. Go to “GitHub Pages” section
    c. Select “master branch”
    d. Click “Save”
    Your site is ready to be published at https://linrongbin.github.io/blog/.

Hexo Config

Config File: _config.yml

  1. 改URL

  2. Deploy to the above git repo.
    change hexo _config.yml file to deploy to the blog repository

  3. Theme of blog
    浪费时间找了很多，最后决定用Star 最多的theme，原因是好看且活得久。
    此帖有排名：https://www.zhihu.com/question/24422335
    下载：

使用: 更新_config.yml里面Extensions部分 in _config.yml

  1. 改语言
    要看看你的theme支持的语言，可以查看\themes\next\languages\ 底下的.yml文件名
    注意：有点theme使用zh-Hans
