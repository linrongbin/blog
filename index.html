Setup of blog

Environment

OS： Windows 7 (64 bit)

Node

Git

hexo

Why：Git 和Node 就够了，jelly 要ruby呀 Info:已经包含很多信息，包括操作系统（os）和 Node

Visual Studio Code

Why：3 个平台都支持，很猛，reduce learning curve Version: Help–> About

vscode extensions

How: Extensions –> Search hexo –> vscode-hexo

Git Config

  1. Repository Setup To create a repository named with “blog” Copy this： https://github.com/{yourself}/blog.git
  2. Enable GitPage
  1. Go to “Setting” link
  2. Go to “GitHub Pages” section
  3. Select “master branch”
  4. Click “Save” Your site is ready to be published at https://linrongbin.github.io/blog/.

Hexo Config

Config File: _config.yml 1. 改URL

  1. Deploy to the above git repo. change hexo _config.yml file to deploy to the blog repository

  2. Theme of blog 浪费时间找了很多，最后决定用Star 最多的theme，原因是好看且活得久。 此帖有排名：https://www.zhihu.com/question/24422335 下载：

使用: 更新_config.yml里面Extensions部分 in _config.yml

  1. 改语言 要看看你的theme支持的语言，可以查看 底下的.yml文件名 注意：有点theme使用zh-Hans

发布

solution is to install hexo-deployer-git Credit to https://github.com/hexojs/hexo/issues/1040

  1. 改favicon.ico 制作工具：https://favicon.io/ 验证工具：https://realfavicongenerator.net/

将 favicon.ico 放在站点的 source 目录下就可以。 更改’theme_config.yml’ 文件
